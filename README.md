thebot-pomodoro
===============

A timer to practice a Pomodoro-style time management. Implemented
as a plugin for [TheBot].

Installation
------------

Run `pip install thebot-pomodoro`, then run thebot with additional
plugin `pomodoro`, like that:

    thebot --plugins pomodoro
